Документация по API NeuroSwitch

Последнее обновление: 2025

Базовый URL

https://api.neuroswitch.ru/v1/

Аутентификация

В каждом запросе передавайте заголовок Authorization: Bearer sk-neuroswitch-xxx

Пример (Python)

from openai import OpenAI
client = OpenAI(api_key="sk-neuroswitch-xxx", base_url="https://api.neuroswitch.ru/v1/")
res = client.chat.completions.create(
  model="gpt-4o-mini",
  messages=[{"role":"user","content":"Пример запроса"}],
  max_tokens=400
)
print(res.choices[0].message)

Поддерживаемые модели

  • OpenAI: gpt-4o, gpt-4o-mini, gpt-3.5-turbo и др.
  • Google: Gemini 2.5, Gemini Pro
  • DeepSeek: r1, r1-fast, chat
  • Anthropic: Claude 3.5/3.7
  • Meta: Llama 3/4
  • Perplexity, xAI Grok и др.
