Генерация и редактирование картинок

Создавайте изображения и редактируйте их слоями. Цена зависит от качества и разрешения.

Создание

import OpenAI from 'openai';
import fs from 'fs';
const client = new OpenAI({ apiKey:'sk-neuroswitch-xxx', baseURL:'https://api.neuroswitch.ru/v1/' });

const img = await client.images.generate({
  model: 'gpt-image-1',
  prompt: 'Иллюстрация: парящий город на закате в стиле synthwave',
  size: '1024x1024',
  output_format: 'png'
});
fs.writeFileSync('city.png', Buffer.from(img.data[0].b64_json, 'base64'));

Редактирование

const edit = await client.images.edit({
  model: 'gpt-image-1',
  prompt: 'Добавь неоновую вывеску на переднем плане',
  image: [
    await OpenAI.toFile(fs.createReadStream('city.png'), null, { type:'image/png' })
  ],
});
fs.writeFileSync('city_edit.png', Buffer.from(edit.data[0].b64_json, 'base64'));

Текстовые и визуальные токены тарифицируются отдельно при редактировании.

Генерация речи из текста (TTS)

Голоса студийного качества с управлением стилем и темпом. Цена — за 1 млн символов.

JS пример

import OpenAI from 'openai';
import fs from 'fs';
const client = new OpenAI({ apiKey:'sk-neuroswitch-xxx', baseURL:'https://api.neuroswitch.ru/v1/' });

const audio = await client.audio.speech.create({
  model: 'gpt-4o-mini-tts',
  voice: 'sage',
  input: 'Привет! Это демо синтеза речи от NeuroSwitch.',
  instructions: 'Дружелюбный тон, средняя скорость'
});
const buf = Buffer.from(await audio.arrayBuffer());
await fs.promises.writeFile('voice.mp3', buf);

FAQ

Работаете с бизнесом? Чеки и закрывающие документы?

Да. Оплата на расчётный счёт, полный комплект документов, ЭДО и скидки на объёмы.

Как работает предавторизация по max_tokens?

Перед запросом мы проверяем баланс на максимально дорогой сценарий, исходя из указанного max_tokens. Фактическая стоимость считается по результату генерации. Если не задавать max_tokens — используется максимум контекста модели.

Почему «размышляющие» модели иногда молчат?

Если max_tokens слишком мал, все токены уходят на внутренние рассуждения, а на ответ не остаётся. Рекомендуем ≥ 50000 для reasoning‑семейств.
